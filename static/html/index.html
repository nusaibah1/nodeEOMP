Welcome to PetCo

HTTP Method Path Description
GET / Displays the static home page.

Users

HTTP Methods Path Description
GET /users Retrieves all users from the database
GET /users/:id Retrieves a single user from the database
POST /users/register Adds a single user to the database
PATCH/PUT /users/update/:id Updates a single user
DELETE /users/delete/:id Deletes a single user from the database

Products/items

HTTP Method Path Description
GET /products Retrieves all products from the database
GET /products/:id Retrieves a single product from the database
POST /products/addProduct Adds a single product to the database
PUT/PATCH /products/update/:id Updates a single product record
